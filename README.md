# Superstore Analysis

## 📊 Overview

A data analysis project exploring sales, profit, and performance insights from superstore data using Python and VS Code.

## 🎯 Project Goals

- Analyze sales trends and patterns
- Identify profitable and underperforming categories
- Understand regional and regional performance
- Discover opportunities to improve business growth

## 🛠️ Tools Used

| Category | Tools |
|----------|-------|
| **IDE** | VS Code |
| **Language** | Python |
| **Data Processing** | Pandas, NumPy |
| **Visualization** | Matplotlib, Seaborn |
| **Notebook** | Jupyter Notebook |

## 🔍 Key Analysis Performed

- **Data Cleaning**: Handled missing values, removed duplicates, cleaned data types
- **Sales Analysis**: Total sales, sales trends over time, monthly/quarterly performance
- **Profit Analysis**: Profit margins, profitable vs unprofitable products
- **Category Analysis**: Best-selling categories, category performance comparison
- **Regional Analysis**: Sales by region, state, and city
- **Segment Analysis**: Customer segment performance (Consumer, Corporate, Home Office)
- **Product Analysis**: Top-selling products, product performance insights

## 📈 Key Insights

✅ Data cleaning and preprocessing workflow  
✅ Sales and profit trends over time  
✅ Category and product performance analysis  
✅ Regional sales comparison  
✅ Customer segment insights  
✅ Actionable business recommendations  


## 📝 Dataset

Superstore dataset containing:
- Orders, sales, profit, quantity
- Product categories and subcategories
- Regions, states, cities
- Customer segments
- Order dates and shipping information
